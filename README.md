# 💳 Fraud Detection ML Dashboard

This project is a Machine Learning-based Fraud Detection System deployed using Streamlit.

## 🚀 Features
- Real-time fraud prediction
- Random Forest model
- SMOTE for class imbalance
- Feature importance visualization
- Interactive dashboard

## 🛠 Tech Stack
- Python
- Scikit-learn
- XGBoost
- Streamlit
- Pandas / NumPy

## 📦 Deployment
Deployed using Streamlit Cloud.
